# 🔐 PGP E-mail Simulator Using GPG

## 📌 Project Overview
The **PGP E-mail Simulator Using GPG** is a network security project that demonstrates how to secure email communication using cryptographic techniques. It uses **GNU Privacy Guard (GPG)**, an open-source implementation of the **OpenPGP** standard, to encrypt, decrypt, sign, and verify messages.

This project simulates secure communication between two users (Sender and Receiver) and showcases how confidentiality, integrity, authentication, and non-repudiation are achieved in real-world systems.

---

## 🎯 Objectives
- Understand **public-key cryptography**
- Generate and manage **GPG key pairs**
- Perform **encryption and decryption**
- Implement **digital signatures**
- Demonstrate **secure email communication**

---

## ❗ Problem Statement
Traditional email communication is insecure and vulnerable to:
- Eavesdropping
- Data tampering
- Identity spoofing

This project solves the problem by implementing **PGP-based secure communication** using GPG.

---

## 🧰 Technologies Used
- **Operating System:** Kali Linux / Ubuntu / Windows

- **Standard:** OpenPGP
